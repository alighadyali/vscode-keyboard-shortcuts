### vscode keyboard shortcuts 
```
split window        - ctrl + \
window up           - ctrl + pg up
window dwn          - ctrl + pg dn
focus explorer      - ctrl + 0
switch view         - ctrl + 1 ..2 ..3
browse histroy      - ctrl + tab
prev file           - alt + left
next file           - alt + right
cmd pallet          - ctrl + shift + o
search code         - ctrl + t
go to line          - ctrl + g
jump back           - ctrl + u
begining of file    - ctrl + home
end of file         - ctrl + end
search def          - ctrl +shift + f
peek implementaion  - alt + F12
peek ref            - shift + F12
rename symbol       - F2
view errors         - F8
hover               - ctrl k + ctrl i
select code block   - alt + shift + left/right
open/focus terminal - ctrl + `
split terminal      - shift + 5
move terminals      - alt + left/right
kill terminal       - ctrl + shift + x
max terminal        - ctrl + shift + q
```
